# Instrucciones para pod-cast

## Cómo usar la aplicación

1.  Abre el archivo `index.html` en tu navegador.
2.  Haz clic en el botón del micrófono para activar el grabador.
3.  Usa los controles para grabar, pausar y detener tu audio.
4.  Las grabaciones aparecerán en la lista de "Grabaciones".
5.  Puedes reproducir, renombrar y eliminar tus grabaciones.
6.  Selecciona una grabación de la lista para cargarla en el editor de sonido.
7.  Usa los controles del editor para aplicar efectos, recortar y ajustar tu audio.

## Cómo desarrollar la aplicación

El código está organizado de la siguiente manera:

-   `index.html`: La estructura principal de la aplicación.
-   `style.css`: Los estilos de la aplicación.
-   `js/`: La carpeta que contiene los archivos de JavaScript.
    -   `app.js`: El archivo principal de la aplicación.
    -   `editor.js`: El código para el editor de sonido.
    -   `recorder.js`: El código para el grabador de audio.
    -   `effects.js`: El código para los efectos de audio.
    -   `utils.js`: Funciones de utilidad.
